# BGP At Doors of Autonomous Systems is Simple

`BADASS` (BGP At Doors of Autonomous Systems is Simple) is a team school project of 42.

It is about discovering **GNS3**, **VXLAN** and **BGP EVPN**.

This repository contains:

- [docker](/docker/): The Docker configuration for the project.
- [docs](/docs/): Documentation and personal notes (both in French and English).
- [tutorials](/tutorials/): Detailed walkthroughs to complete the assignment (both in French and English).

It contains also some notes about good practices I discovered, points of the assignment/tutorials I disagree with and helpful resources.
